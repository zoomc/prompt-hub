    45|### P1-1：MarketHeatmap 市场热力图
    46|
    47|**文件**：`components/MarketHeatmap.tsx`
    48|
    49|**功能**：6 个市场的方块热力图，一眼看全局涨跌
    50|
    51|**数据源**：
    52|```typescript
    53|GET /api/v1/market/overview
    54|→ data.indices: Array<{market, name, price, change_percent}>
    55|```
    56|
    57|**设计**：
    58|```
    59|┌──────────────────────────────────────────────┐
    60|│  A 股          │  港股          │  美股      │
    61|│  上证 -1.25%   │  恒生 -1.06%   │  S&P -0.1% │
    62|│  ████████░░    │  ██████░░░░    │  ░░░░░░░░  │
    63|│                │                │            │
    64|│  深证 -0.88%   │  恒科 -0.80%   │  NASDAQ    │
    65|│  ███████░░░    │  ██████░░░░    │  -0.20%    │
    66|│                │                │  ░░░░░░░░  │
    67|├────────────────┼────────────────┼────────────┤
    68|│  创业板 +0.07% │  韩股          │            │
    69|│  ░░░░░░░░░░    │  KOSPI +2.25%  │            │
    70|│                │  ██████████    │            │
    71|└────────────────┴────────────────┴────────────┘
    72|```
    73|
    74|**实现**：
    75|- CSS Grid 布局，3 列 × 2 行
    76|- 方块大小 = 固定比例（非 treemap）
    77|- 颜色映射：`change_pct` → HSL 色相（120° 绿 → 0° 红）
    78|- -3% 到 +3% 线性映射
    79|- 每个方块显示：市场名 + 指数名 + 涨跌幅
    80|- hover 效果：放大 + 显示详情 tooltip
    81|- 点击方块 → 切换 MarketAnalysis Tab
    82|
    83|**TypeScript 接口**：
    84|```typescript
    85|interface HeatmapItem {
    86|  market: string;
    87|  name: string;
    88|  price: number;
    89|  change_pct: number;
    90|  gridArea: string;  // CSS grid-area
    91|}
    92|
    93|function getHeatmapColor(pct: number): string {
    94|  // -3% → red (0°), 0% → gray (210°), +3% → green (120°)
    95|  const clamped = Math.max(-3, Math.min(3, pct));
    96|  const hue = clamped > 0
    97|    ? 120 - (clamped / 3) * 90   // 120° → 30°
    98|    : 210 + (clamped / 3) * 30;  // 210° → 240°
    99|  return `hsl(${hue}, 70%, 45%)`;
   100|}
   101|```
