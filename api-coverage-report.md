## 二、关键发现（v2.0 更新）

### 1. 新浪板块数据是关键突破 🎯

**API**: `https://vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php/Market_Service.getHQNodeStockCount?node=new_FLJK`
**数据**: 163 个概念板块，每个包含：
- 板块名称、代码
- 涨跌幅（%）
- 领涨股名称、代码、涨跌幅

**测试结果**: 完整返回 163 个板块，无反爬限制

**替代**: 完全替代东财被封的 `push2.eastmoney.com/clist` 板块接口

### 2. 东财资金流向历史数据可用

**API**: `https://push2his.eastmoney.com/api/qt/stock/fflow/daykline/get`
**数据格式**:
```
日期,主力净流入,小单净流入,中单净流入,大单净流入,超大单净流入,主力净流入%,小单净流入%,...
```

**测试结果**: 
- 首次请求返回 119 个数据点（完整历史）
- 后续请求间歇性空响应（频率限制）
- **解决方案**: 缓存 + 降级到历史均值

### 3. 东财北向资金和研报 API 不受限制

| API | 状态 | 数据量 |
|-----|------|--------|
| `kamt.rtmin`（北向资金） | ✅ | 241 个数据点 |
| `reportapi`（研报） | ✅ | 3 份研报/次 |

### 4. 基金推荐由 AI 策略分析替代

**无免费 API 可用**，但可通过以下方式实现：
- yfinance 获取基金净值历史
- AI Gateway 生成策略分析（夏普比率、最大回撤、风格漂移检测）
- 用户输入风险偏好 → AI 推荐匹配基金
